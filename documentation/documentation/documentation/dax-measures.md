# ⚙️ DAX Measures Overview

## 🧠 Overview

This project uses a set of reusable and optimized DAX measures to calculate key business KPIs across Finance, Sales, Marketing, Supply Chain, and Executive dashboards.

These measures ensure consistency in calculations and allow dynamic analysis across different filters and dimensions.

---

## 📊 Core Business Measures

### Revenue Metrics
- Net Sales
- Gross Sales
- Discounts

### Profitability Metrics
- Gross Margin
- Gross Margin %
- Net Profit
- Net Profit %

---

## 📈 Time Intelligence Measures

- Year-over-Year (YoY) Growth
- Same Period Last Year (SPLY)
- Month-over-Month (MoM) Growth
- YTD (Year to Date) Analysis

---

## 🎯 Performance Tracking

- Actual vs Target Variance
- Forecast Accuracy
- KPI Achievement %

---

## ⚙️ Design Principles

- Measures are reusable across all report pages
- Centralized KPI logic to avoid duplication
- Optimized for performance
- Built using best practices in DAX

---

## 📌 Business Value

These DAX measures enable:

- Consistent KPI definitions across departments
- Dynamic filtering and slicing of data
- Accurate performance comparison over time
- Enterprise-level reporting standardization
