# 📦 Power BI Desktop File

This folder contains the Power BI Desktop (.pbix) source file for the Enterprise Business Intelligence Dashboard project.

## Contents

- `enterprise-bi-360-dashboard.pbix`

## Requirements

- Microsoft Power BI Desktop (latest version recommended)

## Purpose

The PBIX file contains the complete data model, Power Query transformations, DAX measures, report pages, and visualizations used to build this Business Intelligence solution.
